# seeScrapper



### example
```bash
# Quotes are nessary if title has spaces
./scrapper.py -m "Some trash manga you read"
```

### Usage:
```bash
usage: scrapper.py [-h] -m MANGA

optional arguments:
-h, --help            show this help message and exit
-m MANGA, --manga MANGA
                      The manga to be downloaded
```
